# WellSpring Egypt — Brand Design Spec

Machine-readable brand reference for design tools (Claude Design, etc.).
Apply these tokens and rules to any WellSpring branded output — decks, docs, social, web, proposals.

## File
- `wellspring-brand-spec.json` — full brand spec as design tokens.

## How to use
Paste the contents of the JSON into your design tool's chat/instructions, e.g.:
> "Use this WellSpring brand spec for everything you build — colors, fonts, logo rules."

## Quick reference
- **Primary colors:** WS Blue `#147cab`, WS Green `#a0be55`, White `#ffffff`
- **Heading font:** Fontin Sans / Poppins Bold (Calibri Bold in PowerPoint)
- **Body font:** Myriad Pro / Helvetica / Poppins Regular (Calibri in PowerPoint)
- **Tagline:** Live — from the Inside Out
- **Footer must include:** Address · Hotline 19304 · Website · Social handles

> Note: hex codes are the source of truth for digital; RGB is derived from hex.
> The 2022 PDF's RGB/PMS values did not match the digital hex and are superseded here.
